## ✨ 功能特色

- 📝 **三大科目 · 420 题目** — 科目一（140题）+ 科目二（140题）+ 科目三（140题）
- 🎯 **三种题型** — 单选题、多选题、判断题
- 🏠 **科目选择首页** — 点击卡片进入对应科目练习，两科完全独立
- 🔍 **灵活筛选** — 按章节或按题型定向刷题
- 🔀 **顺序 / 随机切换** — 支持顺序学习或随机抽题
- ⏭️ **智能续做** — 自动从第一道未答题目开始，全部答完则从头复习
- 🔢 **题号跳转** — 输入题号一键跳转，附带合法性校验
- ✅ **即时反馈** — 提交后自动判分、高亮正确/错误答案，并展示解析
- 📊 **实时统计** — 显示进度、正确率、最终得分与评语
- 💾 **本地自动保存** — 答题进度保存在浏览器本地，支持导入 / 导出 JSON
- 📱 **响应式设计** — 手机、平板、电脑均可完美使用
- 🚀 **零依赖** — 纯 HTML + CSS + JS 单文件，离线可用

## 📖 使用说明

1. **选择科目** — 首页点击卡片进入对应科目练习
2. **选择章节** — 顶部下拉框选择「全部章节」或具体章节
3. **选择题型** — 可筛选单选、多选或判断题
4. **点击「开始/重置」** — 生成题目列表，并自动跳到第一道未答题目
5. **答题**
   - 单选题：点击选项直接选中，再点提交
   - 多选题：可点击多个选项后再提交
   - 判断题：二选一
6. **提交答案** — 查看对错与解析
7. **上一题 / 下一题** — 自由导航
8. **跳转** — 在跳转输入框中输入题号，回车或点击 GO 直接跳转
9. **🏠 首页** — 头部左上角按钮，随时返回科目选择页
10. **顺序/随机** — 点击「🔀 顺序」切换出题顺序
11. **完成全部题目** — 查看最终得分与评语

## 🛠️ 题库格式

每道题目的 JSON 结构如下：

```json
{
  "id": "fund-law-001",
  "ch": 1,
  "type": "single",
  "q": "题目内容",
  "opts": ["选项A", "选项B", "选项C", "选项D"],
  "ans": [1],
  "exp": "解析内容"
}
```

- `type`: `single`（单选）| `multi`（多选）| `tf`（判断）
- `ans`: 正确答案索引数组（多选题可包含多个索引）
